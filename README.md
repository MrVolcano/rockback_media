# Rockback Media Website

A modern, responsive static website for Rockback Media - a digital creative agency specializing in web design, branding, video production, and more.

## Features

- Fully responsive design (mobile-first, 320px to 4K screens)
- Clean, modern UI with sleek black background and neon/pastel accents
- Tailwind CSS for styling
- Vanilla JavaScript for interactivity
- WCAG 2.1 AA accessibility compliant
- SEO optimized
- Fast-loading, performance-optimized

## Pages

### Main Pages
- **Home** (`index.html`) - Hero section, services overview, client quotes, CTA
- **About** (`about.html`) - Company story, values, team info
- **Contact** (`contact.html`) - Contact form with validation
- **Blog** (`blog.html`) - Blog placeholder (coming soon)

### Service Pages
- Web Design
- Branding
- Video Production
- Social Media

### Legal Pages
